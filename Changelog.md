# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-16

### Initial Release

#### Features
- AI-powered game generation using Claude Sonnet 4.5
- Support for p5.js and Phaser game engines
- Monaco code editor with syntax highlighting
- Live code highlighting during game execution
- Event filtering (input, collision, scoring, game state, setup)
- Interactive tooltips for educational patterns
- AI chat assistant for code modifications
- Documentation panel with automatic code analysis
- Flow visualization with Mermaid diagrams
- Google OAuth authentication
- MongoDB game persistence
- Export options (JS file, standalone HTML, clipboard)
- Custom theme support for editor
- Real-time code execution and preview

#### Tech Stack
- Frontend: Monaco Editor, p5.js, Phaser, Mermaid.js, Vanilla JS
- Backend: Node.js, Express, MongoDB, Mongoose, Passport.js
- AI: Anthropic Claude API with prompt caching

#### Known Issues
- Mobile compatibility issues
- Rate limiter needs improvement
- Highlighting inconsistencies
- Minor non-performance bugs
